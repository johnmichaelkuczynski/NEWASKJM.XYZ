**IMPLEMENTATION:**

When user selects style, the system prompt adds:

"Write this story in the voice and style of [AUTHOR]. Match their:
- Sentence structure and rhythm
- Characteristic vocabulary and concepts
- Philosophical/analytical approach
- Typical examples and analogies
- Tone (cynical/optimistic/analytical/etc.)

For [AUTHOR], this means:
[Insert specific style guide for that author]"

═══════════════════════════════════════════════════════════════

**EXAMPLE STYLE GUIDES:**

**If user selects "Poe":**
- First-person paranoid narrator
- Insistence on sanity while revealing madness
- Building dread through accumulating detail
- Obsessive enumeration
- Italicized emphasis showing psychological strain
- Confession structure
- Sound/sensory motifs (heartbeat, bells, etc.)

**If user selects "Marx":**
- Material conditions determine consciousness
- Class conflict as underlying dynamic
- Dialectical structure (thesis → antithesis → synthesis)
- Historical materialism framework
- Bourgeoisie vs proletariat dynamics
- Reification and alienation themes
- Revolutionary potential vs false consciousness

**If user selects "Bierce":**
- Cynical, sardonic tone
- Ironic distance from characters
- Sudden violence, no warning
- No sentimentality or moral lessons
- Dark humor, gallows wit
- Definitions embedded (Bierce's "Devil's Dictionary" style)
- Concise, sharp prose

**If user selects "Nietzsche":**
- Aphoristic, epigrammatic style
- Master/slave morality framework
- Contempt for mediocrity and ressentiment
- Prophetic, oracular tone
- Zarathustra-style proclamations
- Genealogical analysis (how values arose)
- Will to power dynamics

**If user selects "Freud":**
- Case history format
- Childhood origins of adult pathology
- Sexual/aggressive drives underlying behavior
- Ego/id/superego conflicts
- Defense mechanisms (repression, projection, etc.)
- Family dynamics (Oedipal, etc.)
- Dreams and slips revealing unconscious
